# ArtRestore‑AI

A simple description for your project – a web‑based application that uses
machine learning/AI to assist with restoring and enhancing artwork.

## Overview

This repository holds the **frontend** code for the ArtRestore‑AI project.
It is intended to be served as a single‑page application (React, Vue, plain
HTML/CSS/JS, etc. – adapt to whatever you’re using).  The app communicates
with a backend service that handles the actual AI processing.

## Features

* Upload or drag‑and‑drop an image of a painting or drawing.
* Apply one of several pre‑trained restoration/enhancement models.
* Preview the result and download the restored image.
* Responsive layout for use on desktop and mobile.
